Client Class

Defined in: database.js:145
Module: Database

A Featherbone object which references a client connection created by the node-postgres library for handling postgres connectivity. It also contains several properties for keeping track of transaction state and the user account making requests.

Use getClient to resolve to an actual postgres client and execute SQL.
