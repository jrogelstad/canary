connect

(
  • referenceOnly
)
Promise

Defined in database.js:184

Resolves to a Connection using the configured postgres user account. If referenceOnly is passed as true then the connection's client value is a reference Client, otherwise the client is an actual postgres client connection. The reference client is necessary to prevent SQL injection within a Datasource request called by services written in the web client and stored in the database, where otherwise a "real" postgres client is used to execute SQL statements for hard coded services such as Services.CRUD, Services.Events and Services.Installer.
